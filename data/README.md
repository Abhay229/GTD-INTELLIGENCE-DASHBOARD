# Data

The Global Terrorism Database (GTD) CSV is a licensed research dataset and
is **not** included in this repository.

1. Request access at https://www.start.umd.edu/gtd/contact/
2. Place the downloaded file at:
   ```
   data/globalterrorism.csv
   ```
3. Run `python train_attack_model.py` once to generate the model artifacts
   in `models/` (already included in this repo if you didn't retrain).

Dataset credit: Global Terrorism Database (GTD), National Consortium for the
Study of Terrorism and Responses to Terrorism (START), University of
Maryland.
